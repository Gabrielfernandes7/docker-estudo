# Introdução

## O que são conteineres?

Containers são ambientes de software leves, portáveis e isolados que empacotam aplicações junto com suas dependências para execução consistente em diferentes plataformas.

## Por que nós precisamos dos conteiners?

Resolvem problemas de inconsistência de ambiente ao trabalhar em equipe ao padronizar ambientes de execução.

## Bare Metal vs VM vs Containers

Bare Metal: executa aplicações diretamente do hardware com desempenho máximo, porém com flexibilidade limitada.

Máquinas virtuais (VMs): utilizam hipervisores para executar múltiplas instências de SO com forte isolamento, porém com maior sobrecarga.

Containers: Compartilham o kernel do SO do host, oferecendo isolamento leve com maior eficiência de recursos. Os containers são mais leves que as VMs.
